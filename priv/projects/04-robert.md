Robert is Nativo Studio's internal Phoenix LiveView project-management system for handling projects, payments, inventory, extras, and PDF invoicing.

* Tracks active projects with real-time payment progress, showing total value, paid amount, balance, and pending balance in a single dashboard.
* Manages project extras, inventory items, calendar events, and expenses alongside core project data.
* Generates PDF invoices and project extras documents from the same interface, with Kamal deployment and SQLite backup support.
